to set up web:
following steps from https://learn.microsoft.com/en-us/windows/dev-environment/javascript/nodejs-on-windows

install nvm (node version manager). this is how we install npm (node package manager)
-https://github.com/coreybutler/nvm-windows/releases/tag/1.1.11
-download nvmsetup.zip and run

install npm
-open cmd with admin rights
-best to install LTS (latest stable)
-nvm list available , to see versions
-nvm install <version> , where version is the LTS version
-once installed, may need to switch to that version
